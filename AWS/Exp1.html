üìù PROCEDURE

  1. Login to AWS Management Console
    • Open https://aws.amazon.com/console and sign in
    • Verify your account is in Free Tier eligible status
  2. Launch an EC2 Instance
    • Navigate to Services ‚Üí EC2 ‚Üí Instances ‚Üí Launch instance
    • Enter instance name: my-first-web-server
    • Select Amazon Linux 2023 AMI
    • Choose instance type: t3.micro (Free tier eligible)
    • Create new key pair: my-first-key
    • Download and securely store the .pem file
    • Under Network settings, configure Security Group:
      • Allow SSH (port 22) from your IP
      • Allow HTTP (port 80) from anywhere (0.0.0.0/0)
    • Review settings and click Launch Instance
    • Wait for instance state to become Running
    • Note down the Public IPv4 address
  3. Connect to the Instance

    Method 1: EC2 Instance Connect (Browser-based - Recommended)

    • Select your instance in EC2 Dashboard
    • Click Connect button
    • Choose EC2 Instance Connect tab
    • Click Connect button
    • Terminal opens in browser

    Method 2: SSH from Windows PowerShell

    • Place your .pem key file in a known directory
    • Open PowerShell and navigate to that folder:
    cd D:\Kislay\AWS
    • Connect using SSH command:
    ssh -i "my-first-key.pem" ec2-user@[Your-Public-IP]
  4. Install Apache Web Server

    Execute the following commands in sequence:

    # Update system packages sudo dnf update -y # Install Apache HTTP Server sudo dnf install httpd -y # Start Apache service sudo systemctl start httpd # Enable Apache to start automatically on boot sudo systemctl enable httpd # Verify Apache is running sudo systemctl status httpd
    ‚úÖ Success Indicator: You should see "active (running)" displayed in green color.
  5. Deploy a Website

    Create and edit the main webpage:

    # Navigate to Apache web directory cd /var/www/html # Create/edit index.html file sudo nano /var/www/html/index.html
    • Add your HTML content (custom website code)
    • Save file: Press Ctrl + X
    • Confirm save: Press Y
    • Exit: Press Enter
    • Set proper permissions:
    sudo chmod 644 /var/www/html/index.html
  6. Access the Website
    • Open a web browser (Chrome, Firefox, etc.)
    • Enter your EC2 instance's Public IPv4 address in the address bar:
    http://[Your-Public-IP-Address] # Example: http://13.51.44.241
    üéâ Success! Your website should now be visible and accessible from anywhere in the world.
